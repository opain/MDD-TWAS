PGC 3 MDD TWAS Analysis Plan

This document is an analysis plan for the Transcriptome-wide Association Study (TWAS) component of the Psychiatric Genomics Consortium 3 (PGC-3) Major Depressive Disorder (MDD) study.

The TWAS field is developing constantly, with new software and reference data being released regularly.

I would suggest the following:

  • Significance threshold
    • Estimate using permtuation procedure
      • The same process that was used to estimate genome-wide significance, except without extrapolating to infinite coverage of transcriptome. TWAS is performed 1,000 times for a random phenotype using the SNP-weights considered in the study, recording the minimum p-value identified for each null TWAS. The 5% quantile of minimum p-values is then calculated to determine the transcriptome-wide significance threshold, as only 1/20 observations under this threshold will occu by chance. This approach accounts for the correlation between genes and the correlation between features representing the same gene across tissues.
    • Alternative: Bonferroni correction for number of features tested
      • This approach is highly conservative as it does not account for the correlation between genes, and features representing the same gene across tissues.
  • TWAS-based Gene-set/-property enrichment
    • Software: TWAS-GSEA
      • Similar to MAGMA but uses predicted expression correlation matrix to account for correlated features.
    • Run analysis across all SNP-weight panels, within tissue groups (postnatal brain, HPA, HPT, Blood), with tissue specific
    • Hypothesis-free gene-ontology sets
    • Brainspan developmental stage enrichment
    • Use Bonferroni multiple testing correction
